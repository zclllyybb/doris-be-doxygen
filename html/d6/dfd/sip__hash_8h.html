SipHash is a fast cryptographic hash function for short strings. Taken from here: https://www.131002.net/siphash/

This is SipHash 2-4 variant.

Two changes are made:

  • returns also 128 bits, not only 64;
  • done streaming (can be calculated in parts).

On short strings (URL, search phrases) more than 3 times faster than MD5 from OpenSSL. (~ 700 MB/sec, 15 million strings per second)
