详细描述

Read and decode parquet column data into doris block column.

Usage:

// Create chunk reader ColumnChunkReader chunk_reader(BufferedStreamReader* reader, tparquet::ColumnChunk* column_chunk, FieldSchema* fieldSchema); // Initialize chunk reader chunk_reader.init(); while (chunk_reader.has_next_page()) { // Seek to next page header. Only read and parse the page header, not page data. chunk_reader.next_page(); // Load data to decoder. Load the page data into underlying container. // Or, we can call the chunk_reader.skip_page() to skip current page. chunk_reader.load_page_data(); // Decode values into column or slice. // Or, we can call chunk_reader.slip_values(num_values) to skip some values. chunk_reader.decode_values(slice, num_values); }
