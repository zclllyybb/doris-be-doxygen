◆ check_required_instructions()

void check_required_instructions ( )

Check SSE and others instructions availability. Calls exit on fail. This function must be called as early as possible, even before main, because static initializers may use unavailable instructions.

You may wonder about strlen. Typical implementation of strlen is using SSE4.2 or AVX2. But this is not the case because it's compiler builtin and is executed at compile time.
