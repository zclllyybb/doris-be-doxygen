◆ UNPACKED_BUFFER_LEN

const int UNPACKED_BUFFER_LEN = 128
staticprotected

A buffer to store unpacked values. Must be a multiple of 32 size to use the batch-oriented interface of BatchedBitReader. We use uint8_t instead of bool because bit unpacking is only supported for unsigned integers. The values are converted to bool when returned to the user.
