◆ nested_place() [1/2]

AggregateDataPtr nested_place ( AggregateDataPtr __restrict  place) const
inlineprotectednoexceptinherited

In addition to data for nested aggregate function, we keep a flag indicating - was there at least one non-NULL value accumulated. In case of no not-NULL values, the function will return NULL.

We use prefix_size bytes for flag to satisfy the alignment requirement of nested state.
