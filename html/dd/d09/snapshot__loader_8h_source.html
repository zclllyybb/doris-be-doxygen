42/*
43 * Upload:
44 * upload() will upload the specified snapshot
45 * to remote storage via broker.
46 * Each call of upload() is responsible for several tablet snapshots.
47 *
48 * It will try to get the existing files in remote storage,
49 * and only upload the incremental part of files.
50 *
51 * Download:
52 * download() will download the remote tablet snapshot files
53 * to local snapshot dir via broker.
54 * It will also only download files which does not exist in local dir.
55 *
56 * Move:
57 * move() is the final step of restore process. it will replace the
58 * old tablet data dir with the newly downloaded snapshot dir.
59 * and reload the tablet header to take this tablet on line.
60 *
61 */
