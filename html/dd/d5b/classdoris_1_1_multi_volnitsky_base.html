◆ hasMoreToSearch()

bool hasMoreToSearch ( )
inline

This function is needed to initialize hash table Returns true if there is nothing to initialize and false if we have something to initialize and initializes it. This function is a kind of fallback if there are many needles. We actually destroy the hash table and initialize it with uninitialized needles and search through the haystack again. The actual usage of this function is like this: while (hasMoreToSearch()) { search inside the haystack with the known needles }
