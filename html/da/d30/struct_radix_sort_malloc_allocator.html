详细描述

Radix sort, has the following functionality: Can sort unsigned, signed numbers, and floats. Can sort an array of fixed length elements that contain something else besides the key. Customizable radix size.

LSB, stable. NOTE For some applications it makes sense to add MSB-radix-sort, as well as radix-select, radix-partial-sort, radix-get-permutation algorithms based on it. Used as a template parameter. See below.
