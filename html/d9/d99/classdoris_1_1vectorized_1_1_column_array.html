详细描述

A column of array values. In memory, it is represented as one column of a nested type, whose size is equal to the sum of the sizes of all arrays, and as an array of offsets in it, which allows you to get each element.

Returns the range of memory that covers all elements of the array. Works for arrays of fixed length values. For arrays of strings and arrays of arrays, the resulting chunk of memory may not be one-to-one correspondence with the elements, since it contains only the data laid in succession, but not the offsets.

Non-constant arrays of constant values are quite rare. Most functions can not work with them, and does not create such columns as a result. An exception is the function replicate (see FunctionsMiscellaneous.h), which has service meaning for the implementation of lambda functions.
