void ALWAYS_INLINE merge_to_via_emplace ( Self &  that,
Func &&  func 
)
inline

Merge every cell's value of current map into the destination map. Func should have signature void(Mapped & dst, Mapped & src, bool emplaced). Each filled cell in current map will invoke func once. If that map doesn't have a key equals to the given cell, a new cell gets emplaced into that map, and func is invoked with the third argument emplaced set to true. Otherwise emplaced is set to false.
