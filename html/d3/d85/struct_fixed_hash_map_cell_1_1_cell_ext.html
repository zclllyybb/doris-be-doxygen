template<typename Key, typename TMapped, typename TState = HashTableNoState>
struct FixedHashMapCell< Key, TMapped, TState >::CellExt

Similar to FixedHashSetCell except that we need to contain a pointer to the Mapped field. Note that we have to assemble a continuous layout for the value_type on each call of getValue().
