详细描述

A utility class for temporarily disabling CPU features. Usage:

{ CpuInfo::TempDisable disabler(CpuInfo::AVX2); // On the previous line, the constructor disables AVX2 instructions. On the next // line, CpuInfo::IsSupported(CpuInfo::AVX2) will return false. SomeOperation(); // On the next line, the block closes, 'disabler's destructor runs, and AVX2 // instructions are re-enabled. }

TempDisable's destructor never re-enables features that were not enabled when then constructor ran.
