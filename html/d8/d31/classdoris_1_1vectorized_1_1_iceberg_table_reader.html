https://iceberg.apache.org/spec/#position-delete-files The rows in the delete file must be sorted by file_path then position to optimize filtering rows while scanning. Sorting by file_path allows filter pushdown by file in columnar storage formats. Sorting by position allows filtering rows while scanning, to avoid keeping deletes in memory.
