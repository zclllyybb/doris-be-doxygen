97 // This interface is the actual implementation of the new version of iterator.
98 // It currently contains two implementations, one is Level0Iterator,
99 // which only reads data from the rowset reader, and the other is Level1Iterator,
100 // which can read merged data from multiple LevelIterators through MergeHeap.
101 // By using Level1Iterator, some rowset readers can be merged in advance and
102 // then merged with other rowset readers.
103 class LevelIterator {
104 public:

117
118 virtual Status next(IteratorRowRef* ref) = 0;

124 bool is_same() const { return _ref.is_same; }

127
