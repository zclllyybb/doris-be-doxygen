template<size_t key_bits>
struct HashTableFixedGrower< key_bits >

When used as a Grower, it turns a hash table into something like a lookup table. It remains non-optimal - the cells store the keys. Also, the compiler can not completely remove the code of passing through the collision resolution chain, although it is not needed. TODO Make a proper lookup table.
