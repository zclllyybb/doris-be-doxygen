\doxysubsection{详细描述}
Memory pool to append something. For example, short strings. Usage scenario\+:
\begin{DoxyItemize}
\item put lot of strings inside pool, keep their addresses;
\item addresses remain valid during lifetime of pool;
\item at destruction of pool, all memory is freed;
\item memory is allocated and freed by large chunks;
\item freeing parts of data is not possible (but look at Arena\+With\+Free\+Lists if you need); 
\end{DoxyItemize}

If the optional \textquotesingle{}start\+\_\+alignment\textquotesingle{} is specified, the start of range is kept aligned to this value.
