\doxysubsection{详细描述}
A wrapper around externally allocated data. 

\mbox{\hyperlink{structdoris_1_1_slice}{Slice}} is a simple structure containing a pointer into some external storage and a size. The user of a \mbox{\hyperlink{structdoris_1_1_slice}{Slice}} must ensure that the slice is not used after the corresponding external storage has been deallocated.

Multiple threads can invoke const methods on a \mbox{\hyperlink{structdoris_1_1_slice}{Slice}} without external synchronization, but if any of the threads may call a non-\/const method, all threads accessing the same \mbox{\hyperlink{structdoris_1_1_slice}{Slice}} must use external synchronization.
