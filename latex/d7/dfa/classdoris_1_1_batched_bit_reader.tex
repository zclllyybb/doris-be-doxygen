\doxysubsection{详细描述}
Utility class to read bit/byte stream. This class can read bits or bytes that are either byte aligned or not. It also has utilities to read multiple bytes in one read (e.\+g. encoded int). Exposes a batch-\/oriented interface to allow efficient processing of multiple values at a time.
