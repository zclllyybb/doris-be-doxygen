struct doris\+::vectorized\+::\+Num\+Comparison\+Impl$<$ A, B, Op $>$}
Comparison functions\+: ==, !=, \texorpdfstring{$<$}{<}, \texorpdfstring{$>$}{>}, \texorpdfstring{$<$}{<}=, \texorpdfstring{$>$}{>}=. The comparison functions always return 0 or 1 (UInt8).

You can compare the following types\+:
\begin{DoxyItemize}
\item numbers and decimals;
\item strings and fixed strings;
\item dates;
\item datetimes; within each group, but not from different groups;
\item tuples (lexicographic comparison).
\end{DoxyItemize}

\mbox{\hyperlink{classdoris_1_1vectorized_1_1_exception}{Exception}}\+: You can compare the date and datetime with a constant string. Example\+: Event\+Date = \textquotesingle{}2015-\/01-\/01\textquotesingle{}.
