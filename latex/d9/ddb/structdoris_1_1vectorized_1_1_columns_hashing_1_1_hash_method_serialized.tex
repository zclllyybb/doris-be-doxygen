struct doris\+::vectorized\+::\+Columns\+Hashing\+::\+Hash\+Method\+Serialized$<$ Value, Mapped, keys\+\_\+pre\+\_\+serialized $>$}
Hash by concatenating serialized key values. The serialized value differs in that it uniquely allows to deserialize it, having only the position with which it starts. That is, for example, for strings, it contains first the serialized length of the string, and then the bytes. Therefore, when aggregating by several strings, there is no ambiguity.
