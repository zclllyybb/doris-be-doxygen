Compares two numbers. Returns a number less than zero, equal to zero, or greater than zero if a \texorpdfstring{$<$}{<} b, a == b, a \texorpdfstring{$>$}{>} b, respectively. If one of the values is NaN, then
\begin{DoxyItemize}
\item if nan\+\_\+direction\+\_\+hint == -\/1 -\/ NaN are considered less than all numbers;
\item if nan\+\_\+direction\+\_\+hint == 1 -\/ NaN are considered to be larger than all numbers; Essentially\+: nan\+\_\+direction\+\_\+hint == -\/1 says that the comparison is for sorting in descending order.
