\doxysubsection{详细描述}
Properties of data type. Contains methods for serialization/deserialization. Implementations of this interface represent a data type (example\+: UInt8) or parametric family of data types (example\+: \mbox{\hyperlink{namespacedoris_1_1vectorized_a927ff84d34a7989fb159d63d6b4eaf04a4410ec34d9e6c1a68100ca0ce033fb17}{Array(...)}}).

Data\+Type is totally immutable object. You can always share them.
