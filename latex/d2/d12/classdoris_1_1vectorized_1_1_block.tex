\doxysubsection{详细描述}
Container for set of columns for bunch of rows in memory. This is unit of data processing. Also contains metadata -\/ data types of columns and their names (either original names from a table, or generated names during temporary calculations). Allows to insert, remove columns in arbitrary position, to change order of columns.
