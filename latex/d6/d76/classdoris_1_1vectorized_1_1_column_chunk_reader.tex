\doxysubsection{详细描述}
Read and decode parquet column data into doris block column. 

Usage\+:

// Create chunk reader \mbox{\hyperlink{classdoris_1_1vectorized_1_1_column_chunk_reader}{Column\+Chunk\+Reader}} chunk\+\_\+reader(\+Buffered\+Stream\+Reader$\ast$ reader,                                tparquet\+::\+Column\+Chunk$\ast$ column\+\_\+chunk,                                Field\+Schema$\ast$ field\+Schema); // Initialize chunk reader chunk\+\_\+reader.\+init(); while (chunk\+\_\+reader.\+has\+\_\+next\+\_\+page()) \{ // Seek to next page header. Only read and parse the page header, not page data. chunk\+\_\+reader.\+next\+\_\+page(); // Load data to decoder. Load the page data into underlying container. // Or, we can call the chunk\+\_\+reader.\+skip\+\_\+page() to skip current page. chunk\+\_\+reader.\+load\+\_\+page\+\_\+data(); // Decode values into column or slice. // Or, we can call chunk\+\_\+reader.\+slip\+\_\+values(num\+\_\+values) to skip some values. chunk\+\_\+reader.\+decode\+\_\+values(slice, num\+\_\+values); \}
