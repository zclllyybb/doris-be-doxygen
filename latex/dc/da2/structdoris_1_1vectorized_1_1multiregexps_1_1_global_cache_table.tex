\doxysubsection{详细描述}
Maps string pattern vectors + edit distance to compiled vectorscan regexps. Uses the same eviction mechanism as the Local\+Cache\+Table for re2 patterns. Because vectorscan regexes are overall more heavy-\/weight (more expensive compilation, regexes can grow up to multiple MBs, usage of scratch space), 1. \mbox{\hyperlink{structdoris_1_1vectorized_1_1multiregexps_1_1_global_cache_table}{Global\+Cache\+Table}} is a global singleton and, as a result, needs locking 2. the pattern compilation is done outside \mbox{\hyperlink{structdoris_1_1vectorized_1_1multiregexps_1_1_global_cache_table}{Global\+Cache\+Table}}\textquotesingle{}s lock, at the cost of another level of locking.
