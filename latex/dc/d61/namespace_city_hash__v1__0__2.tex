\doxysubsection{详细描述}
This is a version of City\+Hash that predates v1.\+0.\+3 algorithm change. Why we need exactly this version? Although hash values of City\+Hash are not recommended for storing persistently anywhere, it has already been used this way in Click\+House\+:
\begin{DoxyItemize}
\item for calculation of checksums of compressed chunks and for data parts;
\item this version of City\+Hash is exposed in city\+Hash64 function in Click\+House SQL language;
\item and already used by many users for data ordering, sampling and sharding. 
\end{DoxyItemize}
