struct doris\+::vectorized\+::\+Number\+Traits\+::\+Result\+Of\+Addition\+Multiplication$<$ A, B $>$}
The result of addition or multiplication is calculated according to the following rules\+:
\begin{DoxyItemize}
\item if one of the arguments is floating-\/point, the result is a floating point, otherwise -\/ the whole;
\item if one of the arguments is signed, the result is signed, otherwise it is unsigned;
\item the result contains more bits (not only meaningful) than the maximum in the arguments (for example, UInt8 + Int32 = Int64). 
\end{DoxyItemize}
