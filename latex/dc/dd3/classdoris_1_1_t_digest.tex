Converts a quantile into a centroid scale value. The centroid scale is nomin\+\_\+ally the number k of the centroid that a quantile point q should belong to. Due to round-\/offs, however, we can\textquotesingle{}t align things perfectly without splitting points and sorted. We don\textquotesingle{}t want to do that, so we have to allow for offsets. In the end, the criterion is that any quantile range that spans a centroid scale range more than one should be split across more than one centroid if possible. This won\textquotesingle{}t be possible if the quantile range refers to a single point or an already existing centroid. 

This mapping is steep near q=0 or q=1 so each centroid there will correspond to less q range. Near q=0.\+5, the mapping is flatter so that sorted there will represent a larger chunk of quantiles.
