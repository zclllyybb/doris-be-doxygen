\doxysubsection{详细描述}
\mbox{\hyperlink{classdoris_1_1_cache}{Cache}} the result of one SQL, include many partition rowsets. Sql \mbox{\hyperlink{classdoris_1_1_cache}{Cache}}\+: The partition ID comes from the partition last updated. Partition \mbox{\hyperlink{classdoris_1_1_cache}{Cache}}\+: The partition ID comes from the partition scanned by query. The above two modes use the same cache structure.
