\doxysubsection{详细描述}
Class that specifies nullable columns. A nullable column represents a column, which may have any type, provided with the possibility of storing NULL values. For this purpose, a \mbox{\hyperlink{classdoris_1_1vectorized_1_1_column_nullable}{Column\+Nullable}} object stores an ordinary column along with a special column, namely a byte map, whose type is Column\+UInt8. The latter column indicates whether the value of a given row is a NULL or not. Such a design is preferred over a bitmap because columns are usually stored on disk as compressed files. In this regard, using a bitmap instead of a byte map would greatly complicate the implementation with little to no benefits.
