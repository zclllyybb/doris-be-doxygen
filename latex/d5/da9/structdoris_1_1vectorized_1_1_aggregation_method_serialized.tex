\doxysubsection{详细描述}
\subsubsection*{template$<$typename TData$>$\newline
struct doris\+::vectorized\+::\+Aggregation\+Method\+Serialized$<$ TData $>$}
Aggregates by concatenating serialized key values. The serialized value differs in that it uniquely allows to deserialize it, having only the position with which it starts. That is, for example, for strings, it contains first the serialized length of the string, and then the bytes. Therefore, when aggregating by several strings, there is no ambiguity.
