\doxysubsection{详细描述}
\mbox{\hyperlink{classdoris_1_1_cache}{Cache}} results of query, including the entire result set or the result set of divided partitions. Two data structures, one is unordered\+\_\+map and the other is a doubly linked list, corresponding to a result node. If the cache is hit, the node will be moved to the end of the linked list. If the cache is cleared, nodes that are expired or have not been accessed for a long time will be cleared.
