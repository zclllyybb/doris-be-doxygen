\doxysubsection{详细描述}
\mbox{\hyperlink{classdoris_1_1_cpu_info}{Cpu\+Info}} is an interface to query for cpu information at runtime. The caller can ask for the sizes of the caches and what hardware features are supported. On Linux, this information is pulled from a couple of sys files (/proc/cpuinfo and /sys/devices)
