\doxysubsection{详细描述}
The runtimefilter is built in the join node. The main purpose is to reduce the scanning amount of the left table data according to the scanning results of the right table during the join process. The runtimefilter will build some filter conditions. that can be pushed down to node based on the results of the right table.
